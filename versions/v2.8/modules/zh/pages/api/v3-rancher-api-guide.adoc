== 如何使用 API

API 有自己的用户界面，你可以从 Web 浏览器访问它。这是查看资源、执行操作以及查看等效 cURL 或 HTTP 请求和响应的一种简单的方法。要访问它：

. 单击右上角的用户头像。
. 单击**账号 & API 密钥**。
. 在 **API 密钥**下，找到 **API 端点**字段并单击链接。该链接类似于 `\https://<rancher_fqdn>/v3`，其中 `<RANCHER_FQDN>` 是 Rancher deployment 的完全限定域名。
